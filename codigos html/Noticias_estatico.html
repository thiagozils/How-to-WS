Ontologia em WEB-SERVICE

A primeira coisa em qualquer ontologia é a descrição de seus conceitos fundamentais. Conceitos são também conhecidas como classes e estes classes são o componente central da maioria das ontologias. Um Conceito explica um grupo de objetos diferentes que partilham características comuns. Por exemplo, os seres humanos partilham características, tais como um conjunto de partes específicas do corpo, a capacidade de falam uma língua etc. A maioria das linguagens de ontologias permitir que o desenvolvedor para definir classes com base destes características. Uma classe pode ser uma subclasse de outra classe; isto significa que, se o classA é uma subclasse de B, então qualquer individual de tipo A também será um indivíduo do tipo B. Um indivíduo representa objetos no mundo real por exemplo. (REKHA,2016). Com esta citação clique e saiba mais sobre o tema.


Referência:
Rekha Bhatia, Manpreet Singh, Privacy Issues in Web Services: An Ontology Based Solution, Procedia Computer Science, Volume 92, 2016, Pages 461-467, ISSN 1877-0509, http://dx.doi.org/10.1016/j.procs.2016.07.368. (http://www.sciencedirect.com/science/article/pii/S1877050916316246)
